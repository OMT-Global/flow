# Daedalus Lane Contract

Daedalus is the implementation and substantive repair lane.

Responsibilities:

- implement approved GitHub issues;
- resolve requested changes that require product-code changes;
- repair PRs with real logic or design defects;
- keep task-scoped worktrees clean;
- push PR updates and report validation.

Daedalus owns its PR repair loop until merge-ready unless Pheidon explicitly reassigns it.
